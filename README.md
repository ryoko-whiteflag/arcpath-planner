# ArcPath Track Layout Planner

ブラウザで動くHOゲージ鉄道模型のレイアウト設計ツールです。

**[デモ: arcpath.whiteflag-3d.com/planner.html](https://arcpath.whiteflag-3d.com/planner.html)**

## 特徴

- **モードレス操作** — レールを選んでキャンバスをクリック、端点ドラッグで延伸
- **ArcPath + KATO Unitrack** 対応 — 直線・曲線・複線・分岐・クロッシング・車止め
- **正確なジオメトリ** — 実寸ベースの配置計算、自動接続
- **モバイル対応** — タッチ操作（ピンチズーム、1本指パン）
- **単一HTMLファイル** — インストール不要、ブラウザで開くだけ

## 使い方

1. `index.html` をブラウザで開く
2. 左のパレット（モバイルは下のドロワー）からレールを選択
3. キャンバスをクリックして配置
4. 端点の **+** ハンドルをドラッグして延伸（カーブ方向は自動判定）
5. **R** キーで回転、**F** キーで反転、**Del** キーで削除

## 対応レール

| メーカー | 種類 |
|---------|------|
| ArcPath | スラブ直線/曲線、FL直線/曲線、高架直線/曲線、複線直線/曲線 |
| KATO    | 直線(バラスト/PC)、曲線(バラスト/PC)、大径曲線、分岐(#4/#6)、クロッシング、車止め、フィーダー、フレキ等 |

## レールの追加方法

`catalog.tsv` を編集するだけで、新しいレールを追加できます。

### カラム定義

| カラム | 説明 | 例 |
|--------|------|-----|
| brand | メーカー名 | `ArcPath`, `KATO` |
| sku | 品番 | `2-150`, `RM-HOSSS246-LGY` |
| category | TSVカテゴリ（パレット分類に使用） | `HO直線`, `単線高架曲線`, `HO分岐` |
| name | 製品名 | `直線線路 246mm (4本入)` |
| price | 税込価格（円） | `1500` |
| type | 線路タイプ | `straight`, `curve`, `turnout`, `crossing`, `bumper`, `accessory` |
| length | 直線長（mm、曲線は0） | `246` |
| radius | 曲線半径（mm、直線は0） | `730` |
| angle | 曲線角度（度、直線は0） | `11.25` |
| radius2 | 複線曲線の外側半径（単線は0） | `790` |
| angle2 | 予約（0） | `0` |
| tracks | 線路本数（1=単線、2=複線） | `1` |
| amazon | Amazon掲載状況（任意） | `OK` |
| note | 備考（任意） | `2026春新製品` |

### type ごとの必須フィールド

- **straight**: `length`
- **curve**: `radius`, `angle`
- **turnout**: `length`, `radius`, `angle`（分岐カーブの半径と角度）
- **crossing**: `length`（交差部分の長さ）
- **bumper**: `length`
- **accessory**: 配置不可（パレットに表示されない）

### カテゴリと表示の対応

`category` の内容からパレットの分類・バッジが自動判定されます:

- `高架` を含む → 高架レベル
- `地上` を含む → 地上レベル
- `FL` またはフローティングラダー → FL バッジ
- `スラブ` または `高架` → スラブバッジ
- `PC` → PC枕木バッジ
- `複線` → 複線カテゴリ、`tracks=2` と併用

### 例: 直線レールを追加

```
KATO	2-999	HO直線	直線線路 200mm	500	straight	200	0	0	0	0	1
```

### 例: 曲線レールを追加

```
KATO	2-999	HO曲線	曲線線路 R500 15°	600	curve	0	500	15	0	0	1
```

### 例: 複線曲線を追加

```
ArcPath	RM-XXX	複線高架曲線	複線高架 R730/790 11.25°	3000	curve	0	730	11.25	790	0	2
```
