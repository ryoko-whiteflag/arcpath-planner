# ArcPath Track Layout Planner

ブラウザで動くHOゲージ鉄道模型のレイアウト設計ツールです。

**[デモ: arcpath.whiteflag-3d.com/planner.html](https://arcpath.whiteflag-3d.com/planner.html)**

## 特徴

- **モードレス操作** — レールを選んでキャンバスをクリック、端点ドラッグで延伸
- **ArcPath + KATO Unitrack** 対応 — 直線・曲線・複線・分岐・クロッシング・車止め
- **正確なジオメトリ** — 実寸ベースの配置計算、自動接続
- **モバイル対応** — タッチ操作（ピンチズーム、1本指パン）
- **単一HTMLファイル** — インストール不要、ブラウザで開くだけ

## 使い方

1. `index.html` をブラウザで開く
2. 左のパレット（モバイルは下のドロワー）からレールを選択
3. キャンバスをクリックして配置
4. 端点の **+** ハンドルをドラッグして延伸（カーブ方向は自動判定）
5. **R** キーで回転、**F** キーで反転、**Del** キーで削除

## 対応レール

| メーカー | 種類 |
|---------|------|
| ArcPath | スラブ直線/曲線、FL直線/曲線、高架直線/曲線、複線直線/曲線 |
| KATO    | 直線(バラスト/PC)、曲線(バラスト/PC)、大径曲線、分岐(#4/#6)、クロッシング、車止め、フィーダー、フレキ等 |
